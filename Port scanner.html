Port Scanner

This project was a pretty small yet interesting adventure, created primarily to explore the functionality and mechanics of network security. It is a Wi-Fi Port Scanner that identifies all open ports within a network and provides an analysis of potential exploits based on data from trusted sources like CISA and Exploit-DB.

The tool was developed entirely in Python, showing capabilities with network analysis and vulnerability detection. The project, though completed in just 3 weeks, offered interesting learning opportunities in network scanning, exploit detection, and leveraging external security databases for analysis.

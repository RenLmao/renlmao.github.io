Website Scanning Tool

Completed on: January 1, 2025

This project was where most of my work went. It is a website graphing tool that aims to create graphs in a 3D tree manner. It uses Unity and HTMLAgilityPack. I mostly chose to do this project to learn graphing theory, as it was a big focal point for this project. I had to create a semi-realistic physics engine that would repel non-cyclical links and make cyclical links gravitate towards each other.

Details

  • Technology: Unity, C#, HTML
  • Duration: 5 months
